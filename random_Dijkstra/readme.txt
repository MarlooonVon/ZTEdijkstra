1、程序运行环境：g++ (Ubuntu 5.4.0-6ubuntu1~16.04.4) 5.4.0

2、程序运行步骤：$ cmake .
		 $ make
		 $ ./random_Dijkstra
   接下来可按命令行提示输入
   1)选择样例：输入0选择case0.txt，输入1选择case1.txt;
   2)调用随机次数：整数
   3)想要获得的解类型：输入1选择距离最小，输入2选择点数最少。

3、输入用例格式说明
    //初始节点设为Id=0,结束节点设为最后一个Id
    节点个数  链路数量  必经点个数  必经边数  不可经边数  步数限制
    //链路信息
    起始节点  结束节点  链路代价
     ...                  //若干行                                 
    // 必经点
    必经点1  必经点2  ... //若干个必经点
    //必经边
    起始节点  结束节点
    ...                   //若干条必经边
    //不可经边
    起始节点  结束节点
    ...                   //若干条不可经边

4、case0.txt是官方用例图。
   case1.txt包含500个节点、1250条边、7个必经节点、2条必经边及1条不可经边。
