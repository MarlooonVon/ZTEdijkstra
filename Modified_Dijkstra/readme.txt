1)程序运行环境：g++ (Ubuntu 4.8.4-2ubuntu1~14.04.1) 4.8.4

2)程序运行步骤：$camke .
		$make
		$./versionWANG [参数1] [参数2] [参数3]
	其中：[参数1]--测试用例文件名
	      [参数2]--选择权重最小(1) 或 路径最短(0)
	      [参数3]--选择迭代次数
  
3)测试用例格式说明：
  	//初始节点ID=0，结束节点设为最后一个ID
  	节点个数  链路数量  可经过节点个数
  	（空行）
  	开始节点  结束节点  链路代价
  	...        //若干行
   	（空行）
   	必经点个数
   	（空行）
   	必经点ID
   	...       //若干个,第一个和最后一个点分别为源点和终点。
   	（空行）
   	必经链路数
   	（空行）
   	开始结点 结束节点  //必经的链路
   	...       //若干条
   	（空行）
   	不可走链路数
   	（空行）
   	开始节点 结束节点  //不可走链路
   	...      //若干条

4) case0.txt 是18个点的官方测试用例。
   case1.txt 包含500个点，1250条边，限制步数为60。
   case2.txt 包含160个点，600条边，限制步数为50。
   case3.txt 包含800个点，2994条边，限制步数为50。
